Суперглобальні змінні РНР

Позначення змінної Характеристика Отримане значення
$_SERVER['SERVER_PROTOCOL'] Змінна містить ім'я і версію інформаційного протоколу, який був використаний для запиту.
$_SERVER Інформація про сервери та середовище виконання.
$_GET Асоціативний масив змінних, переданих скрипту через параметри URL.
$_POST Асоціативний масив змінних, переданих скрипту через HTTP POST.
$_FILES Асоціативний масив елементів (файлів), завантажених через HTTP POST.
$_COOKIE Асоціативний масив змінних, переданих скрипту через HTTP Cookies.
$_SESSION Асоціативний масив змінних поточної сесії скрипта.
$_REQUEST Асоціативний масив, що містить дані $_GET, $_POST та $_COOKIE.
$_ENV Асоціативний масив змінних середовища виконання.
$GLOBALS Містить посилання на всі змінні глобальної області видимості скрипта.
